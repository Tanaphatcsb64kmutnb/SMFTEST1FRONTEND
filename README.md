## ติดตั้งโปรเจกต์

1. ติดตั้ง Flutter SDK (ถ้ายังไม่มี) → https://docs.flutter.dev/get-started/install

2. Clone โปรเจกต์นี้:

```bash
git clone https://github.com/Tanaphatcsb64kmutnb/SMFTEST1FRONTEND.git
cd SMFTEST1FRONTEND
```



3.
ติดตั้ง dependencies ที่ใช้ในโปรเจกต์:
```bash
git clone https://github.com/Tanaphatcsb64kmutnb/SMFTEST1FRONTEND.git
cd SMFTEST1FRONTEND
```

4.Run project
```bash
flutter run
```
